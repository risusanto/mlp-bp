mlp-bp.cpp adalah sebuah library yang memudahkan programmer dalam membangun software menggunakan multi
layer perceptron dan backpropagation, yang ditulis dengan bahasa pemrograman C++

Fitur:
1. Dinamis
-Topologi, input value, dan target value dapat diubah sesuai kebutuhan
-Dapat digunakan untuk berbagai kebutuhan pada software yang menggunakan
multi layer perceptron dan backpropagation standar

2. Open Source
-Feel free to contribute!
